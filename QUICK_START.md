# ⚡ Démarrage rapide

## Si Node.js est déjà installé

```powershell
# 1. Aller dans le dossier du projet
cd C:\Users\indomieG\Desktop\portfolio

# 2. Installer les dépendances
npm install

# 3. Lancer le serveur
npm run dev

# 4. Ouvrir http://localhost:3000 dans votre navigateur
```

## Si Node.js n'est PAS installé

1. **Installer Node.js** : [https://nodejs.org/](https://nodejs.org/) (version LTS)
2. **Redémarrer votre terminal**
3. **Suivre les étapes ci-dessus**

---

**Temps estimé : 5-10 minutes**
